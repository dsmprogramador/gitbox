• CONFIGURAÇÕES INICIAIS DO GIT
  • git config --global user.email "user@email.com"
  • git config --global user.name "username"

  • CRIAÇÃO DA SSH KEY NO GITHUB E TRATAMENTO DE ERROS
  • checando chaves existentes: ls -al ~/.ssh
  • criando uma nova chave: ssh-keygen -t ed25519 -C "email do github"
  • adicionando a chave ao agent: ssh-add ~/.ssh/id_ed25519 //ou nome do arquivo de criação da sshkey
  • in case error (Could not open a connection to your authentication agent) start ssh-agent:
    • eval `ssh-agent -s`
    • ssh-add
  • copiando a chave do agent para adicionar ao github: clip < ~/.ssh/id_ed25519.pub //ou nome do arquivo de criação da sshkey
  • chave criada com sucesso no github
  • testar com um git clone git@github.com:dsmprogramador/gitbox.git //ssh criada nesse exemplo

  • COMANDOS INICIAIS
  • git init
  • git status
  • git add
  • git commit -m "Descrição do versionamento"
  • ENVIANDO ARQUIVOS PARA O REPOSITÓRIO NO GITHUB
  • git push -u origin main
